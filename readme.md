## 📌 Deskripsi

Sistem ini dirancang untuk membantu operasional hotel dengan fitur-fitur utama seperti:

- Manajemen reservasi
- Manajemen kamar
- Manajemen user & role
- Operasional housekeeping
- Dashboard admin
- Sistem resepsionis
